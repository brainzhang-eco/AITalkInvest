# 结语：合上行情消俗虑，煮开香茗见初心

周五下午三点半，A股又收盘了。

办公室内依然有人在讨论着某某妖股的起落，有人在懊恼自己又一次“卖飞”了牛股。林韭韭坐在位子上，听着这些熟悉的声音，却觉得自己像个局外人。

他伸手点开手机，不是去看那个让他心惊肉跳的自选股列表，而是利索地长按了几个金融行情App，选择了“卸载”。

“叮咚。”

老亢发来一条微信：“韭韭，普洱泡开了，过来喝一杯。”

林韭韭走进老亢那间中医馆风格的办公室，发现老亢正对着窗外的落日出神。

“亢哥，App删了。”林韭韭坐下来，动作前所未有的轻快，“以前我觉得投资是‘赚2%差价’的快感，现在才明白，那其实是‘拿生命换幻觉’。我把阵摆好了，指数选好了，清单也背熟了。接下来，我是不是该找点别的事儿干干？”

老亢欣慰地给他续了一杯茶，清香袅袅：“这就对了。投资的终点，不是为了让你天天盯着那根红红绿绿的柱子，而是为了让你有尊严地、安稳地离开那个柱子，去陪陪家人，读读好书，或者哪怕只是去街角吃个热腾腾的煎饼果子。”

“懂了。”林韭韭呷了一口茶，“最好的投资，是投资自己的生活。”

老亢哈哈大笑，举起搪瓷缸子：“九九归一，大道至简。韭韭，祝你今晚睡个好觉。”

夕阳洒在茶桌上，两个人的影子拉得很长很长。在这一刻，市场的喧嚣仿佛隔在千万里之外。

**（全书完）**
