# 序言：天台欲纵林韭韭，煎饼点醒老亢头

周五下午三点半，A股收盘的钟声在林韭韭耳边听起来像极了丧钟。

林韭韭，男，二十八岁，某互联网公司资深搬砖工，此刻正站在公司二十二楼的天台上，迎着微凉的晚风，思考一个哲学问题：为什么他每次试图靠“精准做T”赚回那2%的差价时，市场总能反手给他一个10%的耳光？

“明明算好了的啊……”林韭韭手里攥着手机，屏幕上红绿交织的K线图在他眼里已经模糊成了一团乱麻，“买了就跌，卖了就涨。我这手，难道是被大自然加持过的‘反向指路牌’？”

就在他准备对着夕阳长叹一声，甚至想跨过那道半米高的护栏去“看清世界”时，身后传来一阵吸溜茶水的声音，伴随着一股浓郁的肉香味。

“韭韭啊，这天台风大，容易吹乱发型，更关键的是——容易把脑子里那点儿刚攒下来的智慧给吹跑了。”

林韭韭回头一看，是老亢。

老亢是公司的财务顾问，由于名字里带个“亢”字，又总爱讲些“亢龙有悔”之类的玄学道理，大家都管他叫老亢。此时他正蹲在阴凉处，一手端着印有“为人民服务”字样的老式瓷缸子，另一手抓着套热气腾腾的煎饼果子，吃得正香。

“亢哥，你别劝我。”林韭韭眼眶微红，“我悟了，我不适合投资。我这底仓明明是个大公司，我就是想在波段上赚点儿零花钱，怎么就越赚越亏，现在连本金都要‘膝盖斩’了呢？”

老亢不紧不慢地咽下最后一口煎饼，指了指林韭韭手里的手机：“你那不叫投资，你那是‘在雷区里跳踢踏舞’。我问你，刚才你吃的那煎饼，如果是五块钱一套，你觉得贵吗？”

林韭韭愣了一下：“五块？那简直是慈善，我能买一车。”

“那要是五十块一套呢？”老亢又问。

“那摊主肯定疯了，我宁愿去吃肯德基。”

“你看，”老亢吸溜了一口普洱，嘿嘿一笑，“你连个煎饼的贵贱都分得清，怎么一进股市，就光盯着那2%的差价，连自己买的是煎饼还是石头都不知道了呢？你那个‘赚2%就跑’的策略，本质上是想靠猜摊主的脸色来发财，而不是看煎饼值多少钱。”

林韭韭张了张嘴，没说出话来。

“下来吧，别在上面凹造型了。”老亢拍了拍裤子上的渣子，站起身来，“投资这事儿，极简单也极复杂。它既是《孙子兵法》里的排兵布阵，也是菜市场里的讨价还价。你要是真想学，我这儿有几壶好茶，还有几段陈年旧事，咱们慢慢聊。”

林韭韭看着老亢那从容的背影，又看了看手机里那仿佛在嘲笑他的分时图，终于咬咬牙，跨下了台阶。

“亢哥，你等等！你刚才说那煎饼果子，要是摊主想卖给我，我到底该出多少钱盘下他的摊子？”

老亢头也不回地挥了挥手：“这就是我们要聊的‘估值之术’了。走，先去泡茶！”

这一老一少，一硬核一呆萌的投基兵法之路，便从这一口煎饼果子的香气中，正式拉开了序幕。
