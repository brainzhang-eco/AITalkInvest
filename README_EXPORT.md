# 导出说明：如何获得精美 PDF

我已经为您生成了两个版本的全书合集：

1.  **`韭韭归一：老亢给小白的投基兵法.html`**：
    *   **推荐方式**：这是最精美的版本。它包含了专业的排版样式（如宋体/黑体适配、精致的引用块、自动分页）。
    *   **如何操作**：直接用浏览器（如 Chrome, Edge 或 Safari）打开该文件，然后按下 `Ctrl+P` (或 `Cmd+P`) 选择“另存为 PDF”即可。

2.  **`韭韭归一：老亢给小白的投基兵法.md`**：
    *   **推荐方式**：适合使用 VS Code 或 Typora 的用户。
    *   **如何操作**：在编辑器中使用 “Export to PDF” 插件，可以获得原汁原味的 Markdown 风格 PDF。

由于当前环境限制，无法直接在服务器端进行高质量的 PDF 渲染（缺少 Chromium 或 Pandoc 引擎），建议您使用上述 HTML 版本进行本地打印，效果最佳。
